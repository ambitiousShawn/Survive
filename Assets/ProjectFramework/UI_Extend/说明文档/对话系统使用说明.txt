对话系统 v1.0 使用说明：

### 1.使用说明：

创建对话时，在 Resources/SO/ 目录下 右键 Create -> Dialogue -> Create Dialogue 创建一个对话模型。在对话模型中进行对话的编辑，目前可开放模式：
	- 1. Text ： 纯文本对话，在Info中填入对话内容即可。
	- 2. TextWithAvatar： 文本 + 贴图模式，除文本外，在Avatar拖入说话人贴图资源。
	- 3. Options：选项模式，可在Option模型添加玩家的回复选项，并填入跳转位置。
目前仅开放Text功能，后续逐步添加。关于Text模式，对话默认滚动为按照编号逐步滚动：0 -> 1 -> 2 ......

使用该模式时，Pos无需手动设置，点击右上角三个点选择 “序列化位置编号”可自动为Pos字段序列化。（面向线形对话流程）。
如需跳转位置，敬请期待后续Options相关模式。当一段对话结束后，以空结点结尾即可（详细案例见 Resources/SO/ExampleDialogue 的Pos = 4 的结点

！！！切记对话结束以专门的空结点结尾！！！

即Pos为 0 ~ 3 为一段对话，Pos为 5~8为另一段对话，“调用时机”采用文本格式放在同目录下（供给程序在对应位置调用，使用能听懂的人话即可）。
